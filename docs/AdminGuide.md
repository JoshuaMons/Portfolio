# Admin guide

## Inloggen
- Ga naar `/login`
- Log in met je Supabase account
- Je wordt doorgestuurd naar `/admin`

## Projecten beheren
- `/admin/projects`
- Vul een **URL** in voor de modal preview
- Als iframe niet werkt (CSP/X-Frame-Options), gebruik de **thumbnail fallback** in de preview modal op `/projects`

## Logboek beheren
- `/admin/blog`
- Publiceer posts door status op `published` te zetten (en optioneel `published_at`)

## Timeline beheren
- `/admin/timeline`
